　 　 英国 《 防务 新闻 》 2006 年 3 月 17 日 报道 　 近日 ， 美国 日内瓦 航宇 公司 ( Geneva   Aerospace ) 的 一架 " 达科 他 " ( Dakota ) 无人机 ( UAV ) ， 在 美陆军 位于 亚利桑那州 尤马 试验场 成功 完成 了 连续 6 次 自主 着陆 试验 。 这一 试验 是 根据 美海军 航空 系统 司令部 ( NAVAIR ) 和 美 空军 无人机 作战 实验室 ( UAVB ) 所 授予 的 合同 进行 的 ， 目的 是 为 " 金雕 " ( Berkut ) 新型 UAV 的 试飞 降低 风险 。 " 金雕 " 具有 自主 着陆 能力 ， 将 采用 该 公司 已 得到 验证 的 综合 航空 电子 技术 ， 预定 在 今年 上半年 开始 试飞 。 　 　 此次 试验 由 日内瓦 航宇 公司 和 NAVAIR 的 人员 合作 完成 ， 所 采用 的 " 达科 他 " UAV 重约 200 磅 ( 90.7 千克 ) 、 翼展 约 16 英尺 ( 4.88 米 ) 。 该机 还 配备 了 日内瓦 航宇 公司 已 得到 验证 的 指挥 / 控制 / 通信 ( C3 ) 系统 ， 包括 " 任务 技术 " ( missionTEK ) 地面 控制站 ， " 飞行技术 " ( flightTEK ) 飞行 控制 计算机 和 " 链接 技术 " ( linkTEK ) 数据链 计算机 。 该 公司 的 这些 成套 C3 产品 加上 自主 着陆 技术 ， 可 提供 经济 可靠 的 UAV 自主 飞行 解决方案 。 　 　 日内瓦 航宇 公司 的 官员 表示 ， 接下来 他们 将 在 一架 2000 磅 ( 907 千克 ) 级 的 UAV 上 验证 自主 着陆 技术 。 公司 未来 目标 还 包括 UAV 的 自主 着 舰 试验 。 ( 张洋 )
